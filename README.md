# Pentesting Scripts

Collection of custom scripts (Python, Bash, and PowerShell) created for penetration testing, reconnaissance, automation, and exploitation.

### 🛠️ Script Categories

- **Reconnaissance** → Subdomain enumeration, directory brute forcing, port scanning
- **Enumeration** → Service enumeration, user enumeration, vulnerability scanning
- **Exploitation** → Automated exploit helpers, payload generators
- **Post-Exploitation** → Privilege escalation helpers, persistence scripts
- **Automation** → Multi-tool chaining and reporting scripts
- **Utilities** → General useful scripts for daily pentesting workflow

---

### 📁 Languages Used
- Python (Primary)
- Bash
- PowerShell

---

### 📌 Purpose
To automate repetitive tasks, increase efficiency during engagements, and build custom tools when existing ones are insufficient.

### 📈 Current Status
- Building core recon and automation scripts
- Will add more advanced exploitation scripts over time
